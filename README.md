### Trupeer AI · Full-Stack Intern
`January 2026 – Present` · `Bengaluru, India` · `6 Months`

</div>

- **Frontend Revamp** — Sole intern on full Next.js 14 App Router rewrite; architected typed REST client layer, Zod form validation, webhook-driven optimistic UI across the Organization module
- **LLM Integration** — Built Campaign Page Generator feature with Anthropic Claude API; five React landing page templates with CSS custom property color contracts, slug generation, MongoDB persistence
